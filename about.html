Site Functions

This website is a tool for Western Michigan Students to find where a classroom is in Floyd Hall. To use the Find Room page all you need to do is enter your course's CRN number and then your classroom will be on the map where the WMU Logo Appears.

About the Authors

This project was created by Dylan Martin and Oliver Sanchez who are both computer science students at Western Michigan University's College of Engineering and Applied Sciences. The project was created for the classes CS4430 and CS3500 to utilize skills learned in both classes related to web technologies and database management systems. This project uses a web front end to deliver the search function to the user and then the data is stored in a backend database. HTML, CSS, Bootstrap, PHP, and mySQL were used in order to create the application seen.
